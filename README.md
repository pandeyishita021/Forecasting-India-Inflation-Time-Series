This analysis focuses on forecasting inflation in India using time series models such as ARIMA, SARIMA, and SARIMAX. It begins with exploratory data analysis to identify key patterns like trend and seasonality, followed by stationarity testing using the Augmented Dickey-Fuller (ADF) test and necessary transformations. Multiple models are built and compared to evaluate forecasting performance, with SARIMAX emerging as the most accurate. Diagnostic checks, including residual analysis, confirm that the model assumptions are satisfied. Overall, the analysis highlights that inflation dynamics in India are influenced more by short-term shocks than long-term trends, providing a data-driven understanding of inflation behavior.
